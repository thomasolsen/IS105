mkdir = make directory command
cd = change directory command
ls = list directory contents
ls -l = use a long listing format (shows more, owner, size etc.)
ls -la = adds ALL the files in the directory, including hidden.
pwd = print working directory to output the path of the current working directory
pipes = can pass the output from one command to anothers input.
redirection = makes you able to control where the input comes from and output goes.
grep = searches the named input files (or standard if no files are named or a single dash is given as file name) for lines containing a match to the given pattern. Prints the matching lines by default
cat = catenate. It reads data from files, and outputs their contents. Simplest way to display the contents of a file at the command line.
less = sinmple feature-rich command-line file viewer
